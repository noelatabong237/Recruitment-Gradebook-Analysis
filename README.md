# Recruitment Gradebook Analysis — Applicant Scoring System

## Project Overview
A data analysis project designed to evaluate and rank job 
applicants based on performance across multiple recruitment 
tests. The system provides a structured, objective scoring 
framework to support fair and efficient hiring decisions.

## Objectives
- Organise and analyse scores across all recruitment test stages
- Calculate percentage scores per applicant per test
- Identify top-performing candidates using statistical analysis
- Visualise results clearly for hiring decision makers

## Test Categories Analysed
- Safety Test
- Company Psychology Test
- Financial Knowledge Test
- Drug Screening Results

## Tools & Techniques Used
- Microsoft Excel
- Percentage Calculations — score standardisation across tests
- MAX, MIN, AVERAGE — performance benchmarking
- Conditional Formatting — visual flagging of top and weak scores
- Bar Charts — clear comparative visualisation of all applicants
- Sorting and Filtering — ranking candidates by overall performance

## Key Outcomes
- Objective, data-driven ranking of all applicants
- Clear identification of strongest candidates across all tests
- Visual dashboard ready for presentation to hiring managers
- Reusable scoring template adaptable for future recruitment
